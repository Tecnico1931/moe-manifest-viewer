Hot Keys Use
Ctrl or cmd key while clicking Opens manifest link in new viewer
Shift + Click On a stream level manifest in the master level to download it
Shift + Click On the copy url icon to uri encode it before copying
t Opens up new viewer
Shift + t Opens the last closed viewer, until there are none left
Variable 0=False 1=True Use
showVideo=1 Boolean: Turns Video on
muted=1 Boolean: Mutes all Videos
showMetrics=1 Boolean: Turns Metric Panel on
xhrCredentials=1 Boolean: Turns XHRCredentials on
automaticPolling=1 Boolean: Turns Automatic Polling on
pollInterval=4 Number: Sets number of seconds for Automatic Polling
showStallDetector=1 Boolean: Turns Stall Detector Panel on
globalToken=1 Boolean: Turns Global Token on
useHlsjs=1 Boolean: Turns on Hls.js for Safari and Edge
showHlsLogs=1 Boolean: Turns Hls Logs Panel on
scrolling=scrolling options String: 'autoScroll', 'tailManifest', 'none'. Sets scrolling options to what you pick
url=http%3A%2F%2Fstream.m3u8 URI encoded url to load initally
relm=1 Boolean: Shows button to load manifest in RELM
Feature Description
Metrics Panel Turns on panel that will display the video metrics such as bitrate, discontinuities, ads...etc.
XHR Credentials Adds the withCredentials attribute on the XML HTTP Requests for manifests
Automatic Polling Polls the manifest in the viewer every `x` seconds where `x` is the pollInterval
Poll Interval Interval to poll when Automatic Polling is turned on, (in seconds)
Stall Detector Turns on panel that identifies if the stream level manifest are not updating as expected
Global Token Ability to add a global token that is saved and can be reused
Use Hls.js Playback via hls.js, dash.js, or native player
Show Hls Logs Turns on panel that will display Hls Logs
